# Docker: Быстрый старт

Этот гайд поможет тебе поднять инфраструктуру проекта с нуля за считанные минуты.

## 1. Установка
Для Mac M1/M2/M3/M4 скачай **Docker Desktop**.

> [!IMPORTANT]
> При установке убедись, что включен **Apple Virtualization Framework** и **virtiofs** (Settings -> General). Это ускорит работу в 3-10 раз.

## 2. Проверка работы
Открой терминал и введи:
```bash
docker --version
docker compose version
```

## 3. Твой первый контейнер
Давай запустим Qdrant, чтобы убедиться, что всё работает:
```bash
docker run -p 6333:6333 -p 6334:6334 \
    -v $(pwd)/qdrant_storage:/qdrant/storage:z \
    qdrant/qdrant
```
- `-p`: Проброс портов (локальный:контейнерный).
- `-v`: Монтирование папки для вечного хранения данных.

## 4. Базовые команды (Чит-лист)

| Команда | Что делает |
| :--- | :--- |
| `docker ps` | Список запущенных контейнеров |
| `docker images` | Все скачанные образы |
| `docker logs <id>` | Посмотреть, что пишет приложение внутри |
| `docker stop <id>` | Мягкая остановка |
| `docker rm -f $(docker ps -aq)` | Тотальная очистка (осторожно!) |
