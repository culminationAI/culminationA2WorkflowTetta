## 🚀 Зачем это проекту?
В «Интерпретации» реранкер выполняет роль **Верификатора Памяти**. Когда герой пытается что-то вспомнить, Qdrant находит сотни похожих моментов, а BGE-реранкер выбирает те 3-5, которые действительно определяют текущую сцену. Это делает поведение героя последовательным и реалистичным.
