## 🚀 Почему E5?
В проекте «Интерпретация» E5 служит фундаментом системы **GraphRAG**. Она обеспечивает семантическую связность между разрозненными фактами в базе знаний Qdrant, позволяя агентам находить релевантные воспоминания героя по смыслу, а не по ключевым словам.
